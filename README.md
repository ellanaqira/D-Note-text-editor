# D-Note text editor

![img icon](prev_img/dnote_icon.png)

A simple text editor application built with Python.

Quick info: All image assets in this application are made by myself.

## Feature include:
#### File
- writing a text file
- save your file
- open a text file
- create a new file 

#### Edit
- copy
- paste
- cut
- select all
